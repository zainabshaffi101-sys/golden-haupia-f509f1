# Zainab Shaffi — Operations & Executive Support Specialist Portfolio

A premium personal brand website for Zainab Shaffi, an Operations & Executive Support Specialist who helps founders, executives, and growing businesses create operational clarity and build systems that scale.

## Key Technologies

- **TanStack Start** — React-based full-stack framework
- **Tailwind CSS v4** — Utility-first styling
- **Netlify** — Deployment and hosting platform
- **Inter** — Typography (Google Fonts)

## Project Structure

The site is a single-page portfolio with smooth scroll navigation. All sections live in `src/components/sections/` and are assembled in `src/routes/index.tsx`.

## Running Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
